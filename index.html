The n queens puzzle is the problem of placing n chess queens on an n√ón chessboard so that no two queens threaten each other; thus, a solution requires that no two queens share the same row, column, or diagonal.

Generate a random solution

Click the start button in the dashed box in order to start the algorithm and watch it finding a random solution to the queens problem for a board with the given size.

How the algorithm works

The algorithm works as follows. A queen is placed on a random position at the board. All cells that are attacked by this queen, get marked as such.

On the remaining cells, again a queen is being placed at a random position, the cells that are attacked by the queen get marked.

When the algorithm reaches a point where there are less remaining un attacked fields than there are queens to be placed on the board, the algorithm backtracks the most recent step.

That is, removing the last recent placed queen and unmark all cells that were attacked by this queen. Note that that a cell still might be marked as being attacked by another queen which is still on the board.
